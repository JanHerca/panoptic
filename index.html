Panoptic

Panoptic lets you interactively browse giga-pixel stereoscopic panoramas and high-resolution planetary surfaces on your laptop display, on your 3D TV, or with your Oculus Rift.

The common thread linking panoramas and planets is their spherical nature.
A panorama is a sphere viewed from the inside, and a planet is a sphere viewed from the outside.

Getting Panoptic

Panoptic is open source. It binds the Spherical Cube Map rendering library with the Thumb virtual reality application framework.

These pre-compiled binaries statically link the Oculus SDK:

  • Panoptic-20140402 — Windows
  • Panoptic-20140402 — OS X
  • Panoptic-20140402 — Linux 64-bit

Installation is trivial. The executable is self-contained. Just place it in the same directory as the image data that you download next. This will ensure that it can find the data when needed.

Getting Images

Ten example scene definitions are bundled with Panoptic. The image data used by these definitions must be downloaded separately. Panoptic is a tool for browsing very large images, so its inputs are large. Several gigabytes of downloads are necessary to get the full effect of all ten examples. Start with the panoramas (1.1 GB via HTTP or torrent), then work your way over to the Moon (22GB via HTTP or torrent).

Running Panoptic

Download and unzip one or both of the example packages. Start the application normally.

  • The About tab gives basic usage information.
  • The Config tab lets you select a display configuration.
  • The Data tab lets you load imagery.
    • Select and load 0-Panoptic-Example-Moon.xml for the lunar examples.
    • Select and load 1-Panoptic-Example-Panoramas.xml for the panorama examples.
